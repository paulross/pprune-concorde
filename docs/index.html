Concorde Re-mixed

There is a great thread on pprune that contains a fascinating discussion from experts about Concorde. The thread has nearly 2000 posts and around 100 pages. Naturally enough it is ordered in time of each post but since it covers so many subjects it is a little hard to follow a particular subject.

Here I have reorganised the original thread by subject semi-automatically using Python. Any post that refers to a subject is included in a page in the original order of the posts. Posts that mention multiple subjects are duplicated appropriately. I have not changed the content of any post and this includes links and images. Each post is linked to the original so that you can check ;-)
